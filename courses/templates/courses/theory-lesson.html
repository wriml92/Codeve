[이론 LLM 내용 꺼내오기] 변수란 무엇일까?

[개념 소개]

변수(Variable)는 프로그래밍에서 데이터를 저장하고 이름을 붙여서 사용하는 공간이에요. 마치 택배를 보낼 때 물건을 담는 박스와 같다고 생각하면 돼요. 박스에는 물건(데이터)를 담고, 그 박스에 이름(변수명)을 붙여서 나중에 쉽게 꺼내 쓸 수 있어요.

[비유]

저금통에 돈(데이터)를 넣고, 필요할 때 꺼내 쓰는 것과도 같아요. 저금통 이름을 변수로 설정하면, 쉽게 저장하고 가져올 수 있답니다. 

money_box = 1000

[핵심 포인트]

변수의 규칙

  • 변수 이름은 문자와 숫자를 사용할 수 있지만, 숫자로 시작할 수 없어요.
    • - 올바른 예시: 'name1', 'score'
    • - 잘못된 예시: '1name'
  • 대소문자를 구분해요.
    • - 'Name'과 'name'은 서로 다른 변수예요.
  • 변수 이름에 공백이 들어갈 수 없고, 대신 '_'(언더바)를 사용해요.
    • - 예시: 'my_name', 'user_score'
  • 예약어는 변수명으로 사용할 수 없어요.
    • - 예약어: 'for', 'if', 'while', 'class' 등

변수에는 다양한 데이터를 담을 수 있어요:

  • 숫자: 'age = 25'
  • 문자열: 'name = "Alice"'
  • 논리값: 'is_student = True'
  • 리스트: 'scores = [90, 85, 80]'

주의사항

  • 변수는 사용하기 전에 반드시 값을 할당해야 해요. (값을 할당한다는 의미에 대해 알려주면 좋을듯)
  • 변수 이름은 의미있게 지어주세요 ('x', 'y' 보다는 'age', 'name').
  • 대소문자를 정확히 구분해서 사용해야 해요.
  • 예약어를 변수명으로 사용하면 오류가 발생해요.
  • 변수에 저장된 값은 언제든 변경될 수 있어요.
