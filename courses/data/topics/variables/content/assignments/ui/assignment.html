아래의 간단한 파이썬 코드를 실행해보세요. 그런 다음에, 변수 'greeting'과 'name'의 값을 바꿔 다른 인사말을 출력해보세요. 코드를 조금씩 수정하면서 출력 결과가 어떻게 바뀌는지 관찰해보세요.

테스트 케이스:

입력: greeting = 'Hello' name = 'Alice' print(greeting + ', ' + name + '!')

출력: Hello, Alice!

입력: greeting = 'Good morning' name = 'Bob' print(greeting + ', ' + name + '!')

출력: Good morning, Bob!

입력: greeting = 'How are you doing' name = 'Charlie' print(greeting + ', ' + name + '?')

출력: How are you doing, Charlie?
