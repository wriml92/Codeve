VSCode 시작하기

  1. VSCode 실행
    • • Windows: 시작 메뉴에서 'Visual Studio Code' 검색
    • • Mac: Spotlight(⌘ + Space)에서 'Visual Studio Code' 검색
  2. 새 파일 만들기
    • • 단축키: Windows는 Ctrl+N, Mac은 ⌘+N
    • • 또는 상단 메뉴 File > New File 클릭
  3. 파일 저장하기
    • • 단축키: Windows는 Ctrl+S, Mac은 ⌘+S
    • • 파일 이름: lists_practice.py 으로 저장
    • • 저장 위치: 바탕화면이나 문서 폴더 추천

실습 내용

🎯 실습 목표

이번 실습에서는 파이썬의 리스트에 대해 배우고, 이를 사용하여 여러 요소를 순서대로 저장하고 인덱스를 통해 접근하는 방법을 실습해보겠습니다.

📝 실습 설명

  1. 파이썬 리스트를 생성하고, 여러 요소를 추가합니다.
  2. 생성한 리스트에서 특정 인덱스의 요소를 가져옵니다.
  3. 가져온 요소를 출력하여 확인합니다.

🎨 예상 결과

이 실습이 완료되면, 터미널에 리스트의 특정 요소가 출력됩니다.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 리스트 생성
my_list = ['apple', 'banana', 'cherry']  # 여기에는 과일 이름이 담긴 리스트를 생성하였습니다.

# 리스트 요소 접근
print(my_list[1])  # 이 코드는 리스트의 두 번째 요소를 선택하여 출력합니다.

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python lists_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • 리스트의 인덱스는 0부터 시작합니다. 따라서 첫 번째 요소를 가져오려면 my_list[0]을 사용해야 합니다.
  • 오타나 대소문자, 띄어쓰기 오류에 주의하세요. Python은 대소문자를 구분하므로, 'my_list'와 'My_List'는 다릅니다.
  • 실행하는 Python 파일이 저장되지 않았다면, 변경한 내용이 반영되지 않습니다. 항상 변경 후에 파일을 저장하세요.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4
