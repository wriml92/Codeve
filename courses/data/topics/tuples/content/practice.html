실습 내용

🎯 실습 목표

이번 실습에서는 튜플(Tuple)에 대해 배워보고, 튜플을 생성하고, 튜플의 요소에 접근하는 방법을 연습해볼 것입니다.

📝 실습 설명

1. 튜플 생성: 괄호() 안에 쉼표로 구분된 여러 값을 넣어 튜플을 생성합니다. 2. 값 출력: 튜플의 특정 인덱스에 있는 값을 출력해봅니다. 3. 값 변경: 튜플의 값은 변경할 수 없다는 것을 확인해봅니다.

🎨 예상 결과

튜플을 성공적으로 생성하고, 튜플의 값에 접근하거나 변경을 시도하면서 그 결과를 확인할 수 있어야 합니다.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 단계 1: 튜플 생성
my_tuple = ("apple", "banana", "cherry")
print(my_tuple)  # ('apple', 'banana', 'cherry')

# 단계 2: 값 출력
print(my_tuple[1])  # 'banana'

# 단계 3: 값 변경(오류 발생)
my_tuple[1] = "blueberry"  # TypeError: 'tuple' object does not support item assignment

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python tuples_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • 튜플의 값 변경: 튜플은 변경 불가능한(immutable) 자료형입니다. 이를 변경하려고 시도하면 오류가 발생합니다.
  • 인덱스 오류: 튜플의 크기보다 큰 인덱스에 접근하려고 하면, 'IndexError: tuple index out of range' 오류가 발생합니다.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4
