튜플(tuple)의 특징 중 올바르지 않은 것은 무엇인가요?

친구들과 모임을 계획하면서, 각자의 이름과 주소를 같이 저장해야 할 때가 있습니다. 이럴 때 파이썬의 튜플(Tuple)을 사용하면 편리합니다. 튜플은 여러 개의 요소를 하나의 변수에 저장할 수 있으며, 한 번 저장된 요소는 변경할 수 없습니다.

다음과 같은 코드를 보면,

friend_info = ('John', '123 Main St')
print(friend_info[0])
print(friend_info[1])

위의 코드를 실행하면 어떤 결과가 나올까요?
