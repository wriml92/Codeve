비전공자인 제인은 튜플(tuple)을 이용하여 여러 값을 하나의 변수에 저장하고자 합니다.

person = ('Jane', 28, 'Engineer')
name, age, occupation = person
print(name, age, occupation)

아래의 코드를 실행해보고, 주어진 튜플(tuple)의 요소를 바꿔가며 출력 결과가 어떻게 바뀌는지 관찰해보세요. 튜플은 여러 값을 순서대로 나열한 컬렉션으로, 값을 변경할 수 없습니다. 이 특성을 이해하기 위해 코드를 조금씩 수정해보고, 각 수정이 프로그램에 어떤 영향을 미치는지 확인해보세요. # 제공된 코드 my_tuple = (1, 2, 3) print('Original tuple:', my_tuple) # 튜플의 첫 번째 요소를 출력 print('First element:', my_tuple[0]) # 튜플의 마지막 요소를 출력 print('Last element:', my_tuple[-1])

당신은 새로 출시된 게임 '튜플타운'의 주민이 되었습니다. 이 게임에서는 주민들이 매일 아침 자신의 기분과 오늘의 날씨를 고려해 하루의 활동을 결정합니다. 당신은 기분과 날씨가 담긴 튜플 'today_info'를 사용해 'activity_decision' 함수를 호출하여 오늘 할 활동을 결정해야 합니다. 'today_info' 튜플은 첫 번째 요소로 '기분 상태'(좋음, 나쁨)를, 두 번째 요소로 '날씨 상태'(맑음, 흐림, 비)를 담고 있습니다. 함수 'activity_decision'은 오늘의 정보를 바탕으로 하루의 활동을 문자열로 알려줄 것입니다.

테스트 케이스:

입력: today_info = ('좋음', '맑음') activity_decision(today_info)

출력: '산책하기'

입력: today_info = ('나쁨', '비') activity_decision(today_info)

출력: '집에서 영화 보기'
