VSCode 시작하기

  1. VSCode 실행
    • • Windows: 시작 메뉴에서 'Visual Studio Code' 검색
    • • Mac: Spotlight(⌘ + Space)에서 'Visual Studio Code' 검색
  2. 새 파일 만들기
    • • 단축키: Windows는 Ctrl+N, Mac은 ⌘+N
    • • 또는 상단 메뉴 File > New File 클릭
  3. 파일 저장하기
    • • 단축키: Windows는 Ctrl+S, Mac은 ⌘+S
    • • 파일 이름: files_practice.py 으로 저장
    • • 저장 위치: 바탕화면이나 문서 폴더 추천

실습 내용

🎯 실습 목표

파이썬에서 파일을 다루는 방법을 배워봅시다. 텍스트 파일을 생성하고, 그 안에 텍스트를 쓰고, 다시 그 텍스트를 읽어오는 실습을 진행합니다.

📝 실습 설명

  • 파이썬의 내장 함수 open()을 사용해 파일을 생성하고 텍스트를 저장합니다.
  • 생성된 파일을 다시 열어 텍스트를 읽어옵니다.
  • 읽어온 텍스트를 출력합니다.

🎨 예상 결과

터미널에 파일에 작성한 텍스트가 정상적으로 출력되어야 합니다.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 파일을 쓰기 모드('w')로 열고, 'file'이라는 변수에 할당
file = open('test.txt', 'w')

# 파일에 텍스트 쓰기
file.write('Hello, Python!')

# 파일 닫기
file.close()

# 파일을 읽기 모드('r')로 다시 열기
file = open('test.txt', 'r')

# 파일 내용 출력
print(file.read())

# 파일 닫기
file.close()

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python files_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • 파일 경로 오류: 파일을 열 때 파일의 정확한 위치를 지정해주어야 합니다. 현재 디렉토리에 있는 파일을 열 경우 파일 이름만 적어주면 됩니다.
  • 파일 모드 오류: 파일을 열 때 'r' (읽기), 'w' (쓰기), 'a' (추가) 등의 모드를 지정해야 합니다. 모드를 잘못 지정하면 오류가 발생합니다.
  • 파일 닫기 누락: 파일을 열었으면 반드시 닫아야 합니다. 파일을 닫지 않으면 데이터 손실이 발생할 수 있습니다.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4
