파일(file)의 정의는 무엇인가요?

파이썬에서 'recipes.txt'라는 새로운 텍스트 파일을 생성하고, 'Chocolate Cake'라는 레시피 제목을 파일에 쓰려고 합니다. 어떤 코드를 사용해야 할까요?

다음 코드를 실행하면 파일의 내용을 어떻게 출력할까요?

file = open('hello.txt', 'r')
print(file.read())
file.close()

다음 코드에서 발생하는 오류는 무엇일까요?

file = open('example.txt', 'r')
contents = file.reade()
print(contents)

컴퓨터의 파일 시스템을 책꽂이에 비유한다면, 하나의 파일은 책꽂이에서 무엇과 가장 유사할까요?

문제를 준비 중입니다.

비전공자가 파일을 읽고 내용을 출력하는 프로그래밍을 배웠습니다.

file = open('example.txt', 'r')
content = file.read()
print(content)

위 코드에서 사용된 개념들의 올바른 설명은 무엇일까요?

아래에 제공된 파이썬 코드를 실행해보고, 파일 내용을 읽는 방법을 관찰해보세요. 그런 다음, 코드를 조금씩 수정하여 다른 파일을 읽거나 파일 내용을 다르게 출력해보세요.

테스트 케이스:

입력: with open('example.txt', 'r') as file: content = file.read() print(content)

출력: example.txt 파일의 내용이 출력됩니다.

입력: with open('example.txt', 'r') as file: lines = file.readlines() for line in lines: print(line.strip())

출력: example.txt 파일의 각 줄이 한 줄씩 출력됩니다, 줄바꿈 문자는 제거됩니다.

테스트 케이스:

출력: 파일 hello.txt이 생성되었고, 내용이 작성되었습니다.

입력: filename = 'greeting.txt' text_to_write = 'Hello, welcome to the file writing world!' # 파일을 쓰기 모드로 열기 with open(filename, 'w') as file: # 여기에 코드를 작성하세요 print(f'파일 {filename}이 생성되었고, 내용이 작성되었습니다.')

출력: 파일 greeting.txt이 생성되었고, 내용이 작성되었습니다.
