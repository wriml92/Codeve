함수(function)에 대한 설명으로 가장 적절한 것은?

함수는 일련의 코드를 하나의 블록으로 묶는 방법입니다. 예를 들어, 카페에서 주문을 받는 것을 함수로 생각해볼 수 있습니다. 카페에서 주문을 받으면, 원하는 커피를 만들고 그 커피를 내어주는 일련의 과정이 실행됩니다.

다음 파이썬 코드를 보고 질문에 답하시오.

def make_coffee(order):
print('Making', order, '...')
return 'Your', order, 'is ready!'
