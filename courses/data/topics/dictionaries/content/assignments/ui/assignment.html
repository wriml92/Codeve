파이썬에서 'dictionary'의 정의는 무엇인가요?

당신은 파이썬을 사용하여 온라인 쇼핑몰에서 판매되는 상품들의 가격을 기록하고 관리해야 합니다. 각 상품의 이름을 키(key)로, 그 상품의 가격을 값(value)으로 저장하려고 합니다. 어떤 방법이 가장 적절할까요?

다음 코드를 실행하면 어떤 결과가 출력될까요?

info = {'name': 'Alice', 'age': 25}
print(info['name'])

다음 코드에서 발생하는 오류는 무엇일까요?

favorite_colors = {'Alice': 'blue', 'Bob': 'green', 'Charlie': 'red'}
color = favorite_colors['Dave']
print(color)
