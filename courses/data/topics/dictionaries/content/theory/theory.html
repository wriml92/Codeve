개념 소개: 딕셔너리

생일 파티에 참석하려고 하는데, 친구들의 취향을 알아야 한다고 상상해봅시다. 각 친구의 이름을 사용하여 그들이 좋아하는 음식을 알 수 있다면 훨씬 편리할 것입니다, 정확히 이런 상황에서 파이썬의 딕셔너리(Dictionary)를 사용할 수 있습니다. 딕셔너리는 키(Key)와 값(Value)의 쌍으로 구성된 데이터 구조입니다. 우리의 경우, 친구의 이름이 키가 되고, 그들이 좋아하는 음식이 값이 됩니다.
딕셔너리는 다음과 같이 세 가지 방식으로 설명할 수 있습니다:

  1. 키를 사용하여 값을 찾을 수 있는 리스트와 같습니다.
  2. 실생활의 사전과 같습니다. 단어(키)를 찾아 그 의미(값)를 알아내는 것처럼, 키를 통해 값을 찾을 수 있습니다.
  3. 키와 값이 연결된 데이터 쌍입니다. 키는 고유하며, 값을 찾는 데 사용됩니다.
딕셔너리는 정보를 효율적으로 저장하고 검색하는 데 매우 유용합니다. 예를 들어, 학생들의 성적을 저장하거나, 상품의 가격 정보를 관리하는 등의 상황에서 딕셔너리를 사용하게 됩니다.
단어 '딕셔너리'에 대해 처음 들어볼 때, 사전이라는 의미로 받아들이기 쉽습니다. 그래서 키와 값이라는 개념이 낯설 수 있지만, 이는 사전에서 단어와 그 단어의 뜻을 찾는 것과 비슷하다고 생각하면 쉽게 이해할 수 있습니다.

비유

딕셔너리를 이해하는 데 도움이 될 만한 실생활 비유를 몇 가지 생각해봅시다.

  1. 우리가 흔히 사용하는 전화번호부를 생각해보세요. 친구의 이름을 찾아 그에 해당하는 전화번호를 알아내는 것처럼, 딕셔너리에서도 키를 통해 값을 찾아낼 수 있습니다.
  2. 또는, 물품을 보관하는 물품함을 생각해볼 수 있습니다. 각 물품함에는 고유 번호가 있고, 그 번호를 통해 특정 물품을 찾을 수 있습니다. 여기서 물품함의 번호가 딕셔너리의 키, 물품은 값에 해당합니다.
  3. 마지막으로, 도서관의 도서 검색 시스템도 딕셔너리와 비슷합니다. 도서의 제목을 입력하면 그에 맞는 위치 정보를 제공해주는 것처럼, 딕셔너리에서도 키를 통해 값을 얻을 수 있습니다.

# 딕셔너리 예제
party_menu = {'John': 'Pizza', 'Sarah': 'Pasta', 'Mike': 'Burger'}
print(party_menu['John'])  # John의 좋아하는 음식을 출력

# 실행 결과: Pizza

실생활 응용

딕셔너리는 많은 실생활 상황에서 활용할 수 있습니다.

  1. 학교생활: 학생들의 여러 정보(이름, 학번, 성적 등)를 효율적으로 관리하고 검색할 수 있습니다.
  2. 게임: 게임 내 아이템의 이름과 그에 해당하는 정보(가격, 효과 등)를 딕셔너리로 저장할 수 있습니다.
  3. SNS: 사용자 이름과 그에 해당하는 프로필 정보를 딕셔너리로 관리하면 효율적으로 정보를 찾을 수 있습니다.

핵심 포인트

딕셔너리는 키와 값으로 이루어진 쌍을 저장합니다. 키를 통해 값을 빠르게 찾을 수 있어서 정보를 효율적으로 관리할 수 있습니다. 예를 들어, 'my_dictionary = {"apple": "red", "banana": "yellow"}'와 같이 생성할 수 있습니다.
딕셔너리의 키는 고유해야 합니다. 만약 동일한 키가 두 개 있다면, 나중에 추가된 값만이 저장됩니다. 따라서 키는 유일한 값을 갖는 것이 중요합니다.

주의사항

  • 딕셔너리는 순서를 보장하지 않습니다. 딕셔너리에 데이터를 추가한 순서대로 데이터가 저장되지 않을 수 있습니다. 따라서 딕셔너리는 순서가 중요하지 않은 경우에 주로 사용됩니다.
    ❌ 잘못된 예시:
    # 딕셔너리를 리스트처럼 사용하려고 시도
my_dict = {'one': 1, 'two': 2}
print(my_dict[0])  # 'one'을 출력하려고 시도

    # 실행 결과: KeyError

    ✅ 올바른 예시:
    # 딕셔너리의 키를 사용하여 값을 찾음
my_dict = {'one': 1, 'two': 2}
print(my_dict['one'])  # 'one'의 값을 출력

    # 실행 결과: 1
