실습 내용

🎯 실습 목표

Python에서 예외 처리 방법을 이해하고, try-except 문을 직접 작성해봅니다.

📝 실습 설명

본 실습에서는 두 개의 숫자를 입력받아 나누는 프로그램을 작성합니다. 단, 분모가 0이 될 경우를 대비하여 예외 처리를 추가합니다.

🎨 예상 결과

프로그램이 예외를 올바르게 처리하고, 사용자에게 오류 메시지를 보여줍니다.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 사용자로부터 두 개의 숫자를 입력받습니다.
num1 = int(input("첫 번째 숫자를 입력하세요: "))
num2 = int(input("두 번째 숫자를 입력하세요: "))

# try-except 문을 사용하여 예외를 처리합니다.
try:
    result = num1 / num2
    print("결과: ", result)
except ZeroDivisionError:  # 0으로 나누는 예외를 처리합니다.
    print("에러: 0으로 나눌 수 없습니다.")

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python exceptions_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • ZeroDivisionError: 0으로 나눌 수 없는 경우 발생하는 오류입니다. 이 경우에는 예외 처리를 통해 해결할 수 있습니다.
  • IndentationError: 파이썬에서는 들여쓰기가 매우 중요합니다. try와 except 블록 내의 코드는 반드시 들여써야 합니다.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4
