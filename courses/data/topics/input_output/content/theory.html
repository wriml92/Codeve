개념 소개

파이썬에서의 입력과 출력은 프로그램과 사용자 사이의 정보 교환을 의미합니다. 입력은 사용자로부터 정보를 받는 것을, 출력은 프로그램이 사용자에게 정보를 보여주는 것을 말합니다. 이는 마치 우리가 카페에서 주문을 하고, 그 주문을 바탕으로 커피를 받는 과정과 비슷합니다.

비유

파이썬에서의 입력과 출력은 우리가 음식점에서 주문을 하고, 음식을 받는 과정에 비유할 수 있습니다. 먼저, 우리는 메뉴를 보고 원하는 음식을 주문합니다. 이때 주문하는 과정이 바로 '입력'에 해당합니다. 그리고 주문한 음식이 나오면, 그것을 받아 먹습니다. 이때 음식을 받아 먹는 과정이 바로 '출력'에 해당합니다. 

# 사용자로부터 이름을 입력받습니다.
name = input("What's your name? ")

# 입력받은 이름을 출력합니다.
print("Hello, " + name + "!")

# 실행 결과: "What's your name? John" -> "Hello, John!"

실생활 응용

입력과 출력은 프로그래밍에서 굉장히 중요한 개념입니다. 예를 들어, 온라인 쇼핑몰에서 제품을 주문할 때, 우리는 원하는 제품과 수량을 입력하고 결제를 합니다. 그리고 쇼핑몰은 우리의 주문 정보를 바탕으로 주문 확인 메시지를 출력합니다. 이러한 과정이 바로 입력과 출력을 활용한 것입니다.

핵심 포인트

파이썬에서 '입력'은 사용자로부터 정보를 받는 것을, '출력'은 프로그램이 사용자에게 정보를 보여주는 것을 의미합니다.
'input()' 함수를 사용하여 입력을 받고, 'print()' 함수를 사용하여 출력을 할 수 있습니다.

주의사항

  • 'input()' 함수는 항상 문자열 형태로 입력을 받습니다. 따라서 숫자를 입력받을 경우에는 반드시 형변환을 해주어야 합니다. 
    # 잘못된 코드 예시
num = input("Enter a number: ")
print(num * 2)
    # 올바른 코드 예시
num = int(input("Enter a number: "))
print(num * 2)
