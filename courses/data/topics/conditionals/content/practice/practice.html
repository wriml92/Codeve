    • • 저장 위치: 바탕화면이나 문서 폴더 추천

실습 내용

🎯 실습 목표

Python의 조건문(Conditionals)를 이해하고, 조건에 따라 다른 동작을 수행하는 코드를 작성해 봅니다.

📝 실습 설명

Python에서 조건문은 if, elif, else 키워드를 사용합니다. 이 실습에서는 간단한 조건문을 작성해보고, 조건에 따라 다른 동작을 수행하는 코드를 작성해 봅니다.

🎨 예상 결과

본인이 입력한 숫자에 따라 다른 메시지를 출력하는 코드를 작성하고, 실행 결과를 확인합니다.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 사용자에게 숫자를 입력받습니다.
num = int(input("Enter a number: "))

# 입력받은 숫자가 0보다 큰 경우, "Positive number"를 출력합니다.
if num > 0:
    print("Positive number")

# 입력받은 숫자가 0인 경우, "Zero"를 출력합니다.
elif num == 0:
    print("Zero")

# 위의 조건들이 모두 맞지 않는 경우(즉, 숫자가 0보다 작은 경우), "Negative number"를 출력합니다.
else:
    print("Negative number")

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python conditionals_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • 타이핑 오류: Python은 대소문자를 구분하므로 주의깊게 타이핑하세요.
  • 들여쓰기 오류: Python은 들여쓰기를 통해 코드 블록을 구별합니다. if, elif, else 다음 줄은 반드시 들여써야 합니다.
  • 문법 오류: 조건문 뒤에는 항상 콜론(:)이 와야합니다.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4
