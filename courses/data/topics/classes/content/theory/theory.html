개념 소개: 클래스(Class)

사람들이 일상생활에서 빵집을 방문하면, 같은 모양과 맛의 빵을 많이 만들어내는 것을 볼 수 있습니다. 이때 사용되는 '붕어빵 틀'을 생각해보세요. 이 틀은 한번에 많은 붕어빵을 만들 수 있게 해주며, 모든 붕어빵은 틀에 따라 같은 모양과 크기를 가집니다. 프로그래밍에서도 비슷한 역할을 하는 것이 바로 클래스(Class)입니다. 클래스는 프로그래밍에서 객체를 생성하기 위한 '틀' 혹은 '설계도'라고 볼 수 있습니다. 이 클래스를 통해서 객체(인스턴스)를 만들어내면, 그 객체들은 클래스에 정의된 대로 같은 속성과 기능을 가지게 됩니다. 그럼 왜 클래스와 객체를 사용하는 걸까요? 간단히 말하면, 코드를 재사용하고, 구조화하는데 도움이 되기 때문입니다. 예를 들어, '차'라는 클래스를 만들고, 이 클래스를 통해 여러 차량 객체를 만들 수 있습니다. 이 차량 객체들은 각각 다른 색상, 브랜드, 모델 등을 가질 수 있지만, 모두 '차'라는 공통의 특성(속성)과 기능(메소드)을 가지게 됩니다. 이렇게 클래스를 사용하면 코드를 더 효율적으로 관리하고 확장할 수 있습니다. 마치 빵집에서 붕어빵 틀을 사용해서 효율적으로 빵을 만들어내는 것처럼요!

비유

클래스와 객체를 이해하기 위한 비유를 몇 가지 더 생각해볼게요. 첫 번째로, 레고 블록 조립 설명서를 생각해봅시다. 레고 설명서는 블록을 어떻게 조립해야 하는지 알려주는 '틀'이라고 할 수 있습니다. 이 설명서를 따라 블록을 조립하면, 설명서에 따른 레고 모델이 완성되죠. 이때 레고 설명서가 '클래스', 완성된 레고 모델이 '객체'에 해당합니다. 두 번째로, 음식 레시피를 생각해볼 수 있습니다. 레시피는 음식을 만드는 방법을 알려주는 '설계도'입니다. 레시피를 따라 요리를 만들면, 그 요리는 레시피에 따른 특성과 맛을 가집니다. 여기서 레시피가 '클래스', 만들어진 요리가 '객체'에 해당합니다. 

# 클래스 정의
class Car:
    def __init__(self, color, brand):
        self.color = color
        self.brand = brand

    def start(self):
        print(f"{self.brand} 차량이 출발합니다.")
    
# 객체 생성
my_car = Car("Red", "Toyota")

# 메소드 호출
my_car.start()

# 실행 결과: Toyota 차량이 출발합니다.

실생활 응용

클래스와 객체는 프로그래밍의 많은 부분에서 사용됩니다. 예를 들어, 학교에서 학생 정보를 관리하는 시스템을 만든다고 생각해봅시다. '학생'이라는 클래스를 만들고, 이 클래스를 통해 각각의 학생 객체를 생성하면, 각 학생은 자신만의 이름, 학번, 성적 같은 속성을 가질 수 있습니다. 또한, 성적 조회나 정보 수정 같은 기능(메소드)도 가질 수 있겠죠. 또한, 게임을 만드는 경우에도 클래스와 객체를 활용할 수 있습니다. '캐릭터' 클래스를 만들면, 이 클래스를 통해 각각의 게임 캐릭터를 생성할 수 있습니다. 이 캐릭터들은 자신만의 능력치, 장비, 스킬 등을 가지고, 공격하거나 이동하는 등의 기능을 수행할 수 있습니다. 또한, SNS나 메시징 앱에서도 클래스와 객체를 활용합니다. '사용자' 클래스를 통해 각각의 사용자 객체를 생성하고, 이 사용자들은 자신만의 아이디, 프로필, 친구 목록 등의 속성을 가질 수 있습니다. 또한, 메시지를 보내거나 친구를 추가하는 등의 기능을 수행할 수 있습니다.

핵심 포인트

클래스는 객체를 생성하기 위한 '틀' 혹은 '설계도'입니다. 클래스를 통해 객체를 생성하면, 그 객체는 클래스에 정의된 속성과 기능을 가집니다.
클래스와 객체를 사용하면 코드를 재사용하고, 구조화하는데 도움이 됩니다. 이를 통해 코드를 더 효율적으로 관리하고 확장할 수 있습니다.

주의사항

  • 클래스의 메소드를 정의할 때, 첫 번째 인자로 항상 'self'를 포함해야 합니다. 'self'는 현재 객체를 가리키는 키워드로, 메소드 내부에서 객체의 속성에 접근할 때 사용합니다.
    ❌ 잘못된 예시:
    class Car:
    def __init__(color, brand):
        self.color = color
        self.brand = brand
    ✅ 올바른 예시:
    class Car:
    def __init__(self, color, brand):
        self.color = color
        self.brand = brand
