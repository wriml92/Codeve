클래스(class)의 정의는 무엇인가요?

당신은 온라인 서점에서 책을 관리하는 프로그램을 만들고 있습니다. 책마다 제목, 저자, 출판년도와 같은 정보를 저장해야 합니다. 파이썬에서 이 정보를 효율적으로 관리하기 위해 어떻게 해야 할까요?

다음 코드를 실행하면 어떤 결과가 출력될까요?

class Dog:
    def bark(self):
        print('Woof!')

다음 코드에서 발생하는 오류는 무엇일까요?

class Car:
    def __init__(self):
        self.color = 'blue'

프로그래밍에서 '클래스'를 식당의 레시피에 비유한다면, 객체(Object)는 무엇과 가장 유사할까요?

요리를 할 때 레시피에 따라 재료들을 조합하여 요리를 만듭니다. 여러분이 샐러드를 만들고 있다고 가정해봅시다.

class Salad:
   def __init__(self, ingredients):
      self.ingredients = ingredients

위 코드의 결과는 실생활에서 무엇과 가장 비슷할까요?

다음은 Python에서 클래스를 이용해 간단한 계산기를 만드는 코드의 일부입니다.

class Calculator:
    def add(self, a, b):
        return a + b
calc = Calculator()
result = calc.add(4, 5)
print(result)

아래에 있는 Python 클래스 코드를 실행해보고, 다음 지시 사항에 따라 코드를 조금씩 변경하여 결과가 어떻게 달라지는지 관찰해보세요. 이 코드는 간단한 'Book' 클래스를 정의하고 있으며, 책의 제목과 저자를 저장하고 출력하는 기능을 가지고 있습니다. ```python class Book: def __init__(self, title, author): self.title = title self.author = author def print_info(self): print(f'Book: {self.title}, Author: {self.author}') # 책 객체 생성 my_book = Book('The Little Prince', 'Antoine de Saint-Exupéry') # 책 정보 출력 my_book.print_info() ``` 1. 'print_info' 함수 안의 출력 포맷을 변경해보세요 (예를 들어, 'Title -> Author:' 형식으로). 4. 새로운 책 객체를 만들어서, 두 책의 정보를 차례대로 출력해보세요.

당신은 'Pet Simulator' 게임의 개발자입니다. 플레이어가 자신의 애완동물을 생성하고, 이름을 부여하며, 애완동물이 기분이 좋은지 확인할 수 있는 코드를 작성해야 합니다. 이를 위해 'Pet' 클래스를 만들고, 간단한 메소드를 사용하여 애완동물의 상태를 확인하세요. 클래스 내에는 애완동물의 이름을 저장하는 변수와 기분을 확인하는 메소드가 있어야 합니다. 'happy' 메소드는 애완동물이 기분이 좋으면 '행복해!'를, 그렇지 않으면 '...지루해'를 반환해야 합니다. 모든 애완동물은 기본적으로 행복한 상태로 시작합니다.

테스트 케이스:

입력: my_pet = Pet('Buddy') print(my_pet.name) print(my_pet.happy())

출력: Buddy 행복해!

입력: your_pet = Pet('Lucy') print(your_pet.name) print(your_pet.happy())

출력: Lucy 행복해!
