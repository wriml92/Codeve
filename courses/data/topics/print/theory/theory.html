개념 소개

프로그래밍을 시작할 때 가장 먼저 배우는 것은 화면에 무언가를 출력하는 것입니다. 파이썬에서는 print() 함수를 사용하여 텍스트, 숫자 등을 화면에 출력할 수 있습니다. 이는 마치 우리가 종이에 글씨를 쓰는 것과 비슷합니다.

비유

print() 함수는 마치 우리가 칠판에 분필로 글씨를 쓰는 것과 같습니다. 선생님이 칠판에 "안녕하세요!"라고 쓰면 학생들이 그것을 볼 수 있죠. 마찬가지로 print() 함수를 사용하면 컴퓨터 화면에 우리가 원하는 내용을 보여줄 수 있습니다. 

# 간단한 인사말 출력하기
print("안녕하세요!")

# 숫자 출력하기
print(12345)

# 실행 결과: 안녕하세요! 12345

실생활 응용

print() 함수는 프로그램에서 정보를 보여주는 가장 기본적인 방법입니다. 예를 들어, 게임에서 점수를 보여주거나, 계산기 프로그램에서 계산 결과를 보여줄 때 사용됩니다. 또한 프로그램이 잘 작동하는지 확인하기 위해 중간 결과를 확인할 때도 자주 사용됩니다.

핵심 포인트

print() 함수는 괄호 안의 내용을 화면에 출력합니다.
문자열은 따옴표로 감싸서 출력하고, 숫자는 그대로 출력할 수 있습니다.
여러 값을 쉼표로 구분하여 한 번에 출력할 수 있습니다: print("나이:", 20)

주의사항

  • 문자열을 출력할 때는 반드시 따옴표를 사용해야 합니다. 
    # 잘못된 코드 예시
print(안녕하세요)  # 오류 발생!
    # 올바른 코드 예시
print("안녕하세요")
