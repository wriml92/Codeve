print 함수의 정의는 무엇인가요?

Alice는 파이썬으로 프로그래밍을 배우기 시작했습니다. 첫 번째 과제로 'Hello, World!'라는 문장을 화면에 출력해야 합니다. Alice가 코드를 작성할 때 어떤 방법을 사용해야 할까요?

다음 코드를 실행하면 어떤 결과가 출력될까요?

greeting = '안녕하세요!'
print(greeting)

다음 코드에서 발생하는 오류는 무엇일까요?

print('Hello World!')
Print('Goodbye World!')

프로그래밍에서 'print' 명령어의 역할을 일상생활에서의 어떤 상황과 가장 잘 비교할 수 있을까요?

카페에서 친구들과 대화를 나누던 중, 새로 출시된 디저트가 있다는 것을 알게 되었습니다. 카페 메뉴판에는 새 디저트의 이름과 가격이 적혀있습니다. 여러분은 그 디저트를 시켜보기로 결정하고, 이를 카페 종업원에게 알려줍니다.

dessert = '초코 무스 케이크' print('제가 주문할 메뉴는', dessert, '입니다.')

위 코드의 실행 결과는 실생활에서 무엇과 가장 비슷할까요?

비전공자가 프로그래밍 수업에서 파이썬의 기본 출력 기능을 배우고 있습니다.

message = 'Hello, World!'
print(message)

위 코드에서 사용된 개념들의 올바른 설명은 무엇일까요?

다음 제공된 코드를 실행해보세요. 그 다음에는, 제공된 코드의 message 변수에 있는 텍스트를 변경하고 다시 실행해보면서 출력 결과가 어떻게 달라지는지 관찰해보세요.

테스트 케이스:

입력: message = 'Hello, World!' print(message)

출력: Hello, World!

입력: message = '안녕하세요, 파이썬!' print(message)

출력: 안녕하세요, 파이썬!

당신은 방금 프로그래밍을 시작한 신입 마법사입니다. 첫 번째 마법 수업에서는 마법의 주문을 외워서 마법진을 활성화하는 연습을 하게 됩니다. 주문을 올바르게 외우면 마법진이 '반짝'하고 빛나며 활성화됩니다. 당신의 목표는 주어진 마법 주문 변수를 사용해서 마법진을 활성화하는 코드를 작성하는 것입니다. 주어진 변수: magic_spell = 'Abracadabra' 당신이 작성해야 할 코드는 다음과 같습니다: 1. 마법 주문을 화면에 출력하세요. 2. 주문이 화면에 출력되면 마법진이 활성화되며 '반짝!'이라고 표시됩니다. 마법 주문이 올바르게 출력되고 마법진이 활성화되었는지 확인하기 위해 코드를 실행해 보세요!

테스트 케이스:

입력: print(magic_spell) print('반짝!')

출력: Abracadabra 반짝!

입력: print(magic_spell + ', 반짝!')

출력: Abracadabra, 반짝!
