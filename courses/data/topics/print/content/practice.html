VSCode 시작하기

  1. VSCode 실행

  2. 새 파일 만들기
    • • 단축키: Windows는 Ctrl+N, Mac은 ⌘+N

  3. 파일 저장하기
    • • 단축키: Windows는 Ctrl+S, Mac은 ⌘+S
    • • 파일 이름: input_output_practice.py 으로 저장
    • • 저장 위치: 바탕화면이나 문서 폴더 추천

실습 내용

🎯 실습 목표

이번 실습에서는 파이썬의 기본 입출력 함수인 'input()'과 'print()'를 사용해봅니다. 사용자로부터 이름과 나이를 입력받아, 환영 메시지와 내년 나이를 출력하는 프로그램을 작성해보세요.

📝 실습 설명

  1. 먼저 'input()' 함수를 사용해 사용자로부터 이름을 입력받습니다.
  2. 다음으로 'input()' 함수를 사용해 사용자로부터 나이를 입력받습니다. 이때 입력받은 나이는 문자열이므로 숫자로 변환해야 합니다.
  3. 입력받은 이름과 나이를 사용해 환영 메시지를 출력합니다.
  4. 입력받은 나이에 1을 더한 값(내년 나이)을 출력합니다.

🎨 예상 결과

프로그램을 실행하면 다음과 같은 대화가 이루어집니다. (사용자 입력은 ➜ 뒤에 표시되었습니다.) 
What's your name? ➜ John
How old are you? ➜ 20
Hello, John! Welcome!
Next year, you will be 21 years old.

코드 작성하기

아래 코드를 한 줄씩 직접 타이핑해보세요. 복사/붙여넣기 하지 말고, 직접 입력하면서 각 줄이 어떤 의미인지 생각해보세요.

코드를 입력할 때마다 실행해보고, 결과가 어떻게 달라지는지 관찰해보세요.

# 사용자로부터 이름을 입력받습니다.
name = input("What's your name? ")

# 사용자로부터 나이를 입력받습니다. 입력받은 나이는 문자열이므로 숫자로 변환해야 합니다.
age = int(input("How old are you? "))

# 입력받은 이름과 나이를 사용해 환영 메시지를 출력합니다.
print("Hello, " + name + "! Welcome!")

# 입력받은 나이에 1을 더한 값(내년 나이)을 출력합니다.
print("Next year, you will be " + str(age + 1) + " years old.")

💡 코드를 모두 입력한 후:

  1. 실행 결과가 어떻게 될지 예상해보세요
  2. 코드의 각 부분을 변경해보고 결과가 어떻게 바뀌는지 실험해보세요
  3. 자신만의 방식으로 코드를 수정해보세요

코드 실행하기

  1. 터미널 열기
    • • 단축키: Windows는 Ctrl+`, Mac은 ⌘+`
    • • 또는 상단 메뉴 View > Terminal 클릭
  2. 실행 명령어 입력
    python input_output_practice.py
  3. 결과 확인
    • • 터미널에 출력된 결과가 예상 결과와 같은지 확인
    • • 오류가 발생했다면 아래 주의사항을 참고

❗ 주의사항

자주 발생하는 오류

  • 'input()' 함수는 항상 문자열 형태로 입력을 받습니다. 따라서 숫자를 입력받을 경우에는 반드시 형변환을 해주어야 합니다.
  • 문자열과 숫자를 직접 연결하려 할 때 TypeError가 발생할 수 있습니다. 문자열과 숫자를 연결하려면, 숫자를 먼저 문자열로 변환해야 합니다.

도움이 되는 팁

  • 코드를 복사할 때: Windows는 Ctrl+C, Mac은 ⌘+C
  • 코드를 붙여넣을 때: Windows는 Ctrl+V, Mac은 ⌘+V
  • 실행 중인 프로그램 중단: 터미널에서 Ctrl+C (Mac도 동일)
  • 화면 캡처: Windows는 Win+Shift+S, Mac은 ⌘+Shift+4

기본 예제

1. 간단한 메시지 출력하기

print("Hello, Python!")
print("프로그래밍은 재미있어요!")

Hello, Python!
프로그래밍은 재미있어요!

숫자 출력하기

2. 다양한 숫자 출력

실행 결과:

42
3.14
3

여러 값 함께 출력하기

3. 쉼표로 구분하여 출력

print("나이:", 20)
print("이름:", "철수", "학년:", 3)
print("1 + 2 =", 1 + 2)

실행 결과:

나이: 20
이름: 철수 학년: 3
1 + 2 = 3
