VSCode 실습하기

이제 VSCode를 직접 설치하고 사용해보면서 기본적인 기능들을 익혀보겠습니다.

실습 1: VSCode 설치하기

설치 확인하기

  1. VSCode가 이미 설치되어 있다면 이 단계는 건너뛰어도 됩니다.
  2. 설치 여부를 모르겠다면, 아래 방법으로 확인해보세요:
    • Windows: 시작 메뉴에서 'Visual Studio Code' 검색
    • Mac: Spotlight(⌘ + Space)에서 'Visual Studio Code' 검색

설치 과정

  1. 브라우저를 열고 'VSCode 다운로드'를 검색하거나 공식 웹사이트를 방문하세요.
  2. 운영체제에 맞는 다운로드 버튼을 클릭하세요.
  3. 다운로드된 파일을 실행하고 설치를 진행하세요.
  4. 설치가 완료되면 VSCode를 실행해보세요.

실습 2: 기본 기능 익히기

새 파일 만들기

  1. VSCode를 실행하세요.
  2. 다음 중 한 가지 방법으로 새 파일을 만드세요:
    • Windows: Ctrl + N
    • Mac: ⌘ + N
    • 또는 상단 메뉴 [파일 → 새 파일] 클릭
  3. 파일에 다음 내용을 입력해보세요: 
    안녕하세요!
이것은 제가 VSCode로 작성한
첫 번째 파일입니다.

파일 저장하기

  1. 다음 중 한 가지 방법으로 파일을 저장하세요:
    • Windows: Ctrl + S
    • Mac: ⌘ + S
    • 또는 상단 메뉴 [파일 → 저장] 클릭
  2. 파일 이름을 'first_file.txt'로 지정하세요.
  3. 저장 위치는 바탕화면이나 문서 폴더를 추천합니다.

실습 3: 터미널 사용해보기

터미널 열기

  1. 다음 중 한 가지 방법으로 터미널을 열어보세요:
    • Windows: Ctrl + `
    • Mac: ⌘ + `
    • 또는 상단 메뉴 [보기 → 터미널] 클릭
  2. 검은 바탕의 터미널 창이 VSCode 하단에 나타날 것입니다.
  3. 터미널에 다음 명령어를 입력해보세요: 
    echo "Hello from Terminal!"

실습 4: 단축키 연습하기

기본 단축키 연습

  1. 새 파일을 하나 더 만들어보세요. (Ctrl/⌘ + N)
  2. 아무 내용이나 입력해보세요.
  3. 실행 취소를 해보세요. (Ctrl/⌘ + Z)
  4. 다시 실행을 해보세요. (Ctrl/⌘ + Y 또는 Ctrl/⌘ + Shift + Z)
  5. 파일을 저장하지 않고 닫아보세요.

✅ 실습 완료하기

모든 실습을 완료하셨나요? 다음 항목들을 체크해보세요:

  • VSCode가 정상적으로 설치되었나요?
  • 새 파일을 만들고 저장할 수 있나요?
  • 터미널을 열고 간단한 명령어를 실행해보았나요?
  • 기본 단축키들을 사용해보았나요?

위 항목들을 모두 완료하셨다면, VSCode의 기본적인 사용법을 익히신 겁니다! 이제 본격적으로 프로그래밍을 시작할 준비가 되었습니다.
