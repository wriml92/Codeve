VSCode: 코딩을 위한 첫 걸음

VSCode(Visual Studio Code)는 코드 편집기입니다. 마치 한글이나 워드로 문서를 작성하듯이, VSCode로는 프로그램 코드를 작성합니다. 처음에는 낯설 수 있지만, 이 도구 하나로 모든 프로그래밍 언어를 편리하게 다룰 수 있습니다.

1. VSCode 설치하기

VSCode를 사용하기 위해서는 먼저 설치가 필요합니다. 운영체제에 따라 설치 방법이 다르니 아래 순서대로 따라해보세요.

Windows 사용자

  1. 브라우저에서 'VSCode 다운로드' 검색 또는 공식 웹사이트 방문
  2. 'Download for Windows' 버튼 클릭
  3. 다운로드된 설치 파일 실행
  4. '동의함' 체크 후 '다음' 버튼으로 설치 진행

Mac 사용자

  1. 브라우저에서 'VSCode 다운로드' 검색 또는 공식 웹사이트 방문
  2. 'Download for Mac' 버튼 클릭
  3. 다운로드된 .dmg 파일 실행
  4. VSCode 아이콘을 Applications 폴더로 드래그

2. VSCode 처음 실행하기

❗ 처음 실행 시 주의사항

  • Mac에서 처음 실행 시 "확인되지 않은 개발자" 경고가 뜰 수 있어요
  • 이런 경우 [시스템 환경설정 → 보안 및 개인 정보] 에서 "확인 없이 열기" 허용

시작 화면 살펴보기

VSCode를 처음 실행하면 다음과 같은 화면이 보입니다:

  • 시작 페이지: 환영 메시지와 기본 기능 소개
  • 메뉴 바: 파일, 편집, 선택, 보기 등의 메뉴
  • 사이드바: 파일 탐색기, 검색, 확장 기능 등

3. 알아야 할 기본 용어

파일 확장자란?

파일 이름 뒤에 붙는 점(.)과 문자를 확장자라고 해요. 예를 들어, hello.py에서 .py가 확장자입니다. 이는 이 파일이 Python 코드 파일이라는 것을 나타냅니다.

터미널이란?

터미널은 컴퓨터와 대화하는 창입니다. 검은 바탕에 텍스트를 입력하는 것이 낯설 수 있지만, 이곳에서 프로그램을 실행하고 결과를 확인할 수 있습니다.

4. 꼭 알아야 할 단축키

Windows 사용자

  • 새 파일: Ctrl + N
  • 파일 저장: Ctrl + S
  • 터미널 열기: Ctrl + `
  • 실행 취소: Ctrl + Z

Mac 사용자

  • 새 파일: ⌘ + N
  • 파일 저장: ⌘ + S
  • 터미널 열기: ⌘ + `
  • 실행 취소: ⌘ + Z

5. 자주 하는 실수와 해결 방법

파일을 저장하지 않았어요!

파일 이름 옆에 동그라미(⚪)가 있다면, 아직 저장되지 않은 것입니다. 반드시 Ctrl + S(Windows) 또는 ⌘ + S(Mac)로 저장해주세요.

터미널이 안 열려요!

상단 메뉴의 [보기 → 터미널] 을 클릭하거나, Ctrl + `(Windows) 또는 ⌘ + `(Mac)를 눌러보세요.
